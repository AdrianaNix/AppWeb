Electrosa - Tienda Virtual de Electrodomésticos 

¡Bienvenido a Electrosa! Esta es una aplicación web full-stack que simula una tienda virtual (e-commerce) de electrodomésticos. Los usuarios pueden navegar por el catálogo, gestionar su carrito de compras y registrarse para realizar pedidos.

Este proyecto fue desarrollado como parte del curso "Programación de Aplicaciones Web" de la carrera de ingeniería informátia.



Características Principales

Catálogo de Productos:Visualización de todos los electrodomésticos disponibles en la tienda.
Carrito de Compras: Funcionalidad completa para añadir, modificar y eliminar productos del carrito.
Gestión de Cuentas:
    Registro de nuevos clientes.
    Inicio de sesión (Autenticación) de usuarios.
Navegación Intuitiva: Un diseño claro con secciones de Blog, Ofertas y categorías de productos.

Tecnologías Utilizadas

Este proyecto fue construido usando un stack de Java moderno:

Backend: Spring Framework (Spring Boot, Spring MVC)
Frontend (Plantillas):Thymeleaf (para el renderizado dinámico de HTML en el servidor)
Seguridad:Spring Security (para gestionar el login y el registro de forma segura)
Base de Datos: MySQL
Servidor: Apache Tomcat (integrado en Spring Boot)
